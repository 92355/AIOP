# AIOP

**All-In-One Page / Personal Operating Dashboard**

AIOP는 내가 필요한 기능을 하나의 웹페이지에서 관리하기 위한 개인용 운영 대시보드입니다.

이 프로젝트의 초기 목표는 “이 페이지 하나만 켜두면 사고 싶은 것, 자산 계산, 구독, 책 인사이트, 후회 기록, 빠른 메모까지 한눈에 관리할 수 있다”는 사용 경험을 검증하는 것입니다.

---

## 현재 상태

현재 버전은 **프론트엔드 레이아웃 MVP**입니다.

- 백엔드 로직은 아직 구현하지 않았습니다.
- 인증 기능은 아직 없습니다.
- 데이터베이스는 아직 연결하지 않았습니다.
- 외부 API는 아직 사용하지 않습니다.
- AI 기능은 아직 구현하지 않았습니다.
- 모든 데이터는 내부 mock data를 사용합니다.

현재 목적은 실제 기능 구현보다 **화면 구조, 정보 배치, 대시보드 사용 경험**을 먼저 검증하는 것입니다.

---

## 주요 컨셉

> 내가 필요한 모든 것을 하나의 페이지에서 관리한다.

AIOP는 단순한 가계부나 투자앱이 아니라, 개인의 소비 판단, 자산 목표, 구독 관리, 지식 정리, 빠른 메모를 하나로 묶은 개인 운영 페이지를 지향합니다.

---

## 주요 화면

### Dashboard

전체 요약 화면입니다.

- 사고 싶은 항목 요약
- 이번 달 구독비
- 자산으로 커버 가능한 소비 금액
- 최근 인사이트
- 주요 카드형 정보 요약

### Wants

사고 싶은 물건을 관리하는 화면입니다.

- 물건 이름
- 가격
- 카테고리
- 사고 싶은 이유
- 상태
- 구매 판단 점수
- 자산으로 사려면 필요한 금액

### Asset Calculator

“이 소비를 자산 수익으로 감당하려면 얼마가 필요한가?”를 계산하는 화면입니다.

기본 계산 항목:

- 물건 가격
- 목표 기간
- 예상 배당률
- 월 투자 가능 금액
- 필요 자산
- 필요 월 현금흐름
- 예상 구매 가능 기간

### Regret Tracker

“그때 살걸” 기록을 관리하는 화면입니다.

- 관심 있었던 자산 또는 물건
- 당시 가격
- 현재 가격
- 상승/하락률
- 당시 생각
- 현재 결과

현재는 실제 주식 가격 API 없이 mock data로 구성되어 있습니다.

### Subscriptions

월 구독 서비스를 정리하는 화면입니다.

- 서비스명
- 월 구독료
- 카테고리
- 사용 빈도
- 가치 점수
- 유지 / 검토 / 해지 후보 상태

### Book Insights

책, 영상, 아티클, 생각에서 얻은 인사이트를 정리하는 화면입니다.

- 제목
- 출처 타입
- 핵심 인사이트
- 내 행동으로 바꿀 것
- 관련 태그

### Notes / Inbox

빠르게 생각을 기록하는 공간입니다.

- Quick Capture 영역
- 최근 메모
- 태그
- 나중에 Wants, Insights, Subscriptions 등으로 분류될 수 있는 임시 저장소 역할
